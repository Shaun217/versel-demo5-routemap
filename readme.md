# 🗺️ AI Intelligent Route Map (AI 地图路径规划)

这是一个结合了 **LLM (AI 大模型)** 和 **GIS (地理信息系统)** 的旅行规划工具。
它不仅能用文字告诉你怎么走，还能直接在地图上画出真实的道路轨迹。

## ✨ 核心亮点

1.  **AI 智能排序 (TSP Solver)**:
    - 输入任意乱序的景点，AI 会根据地理常识，重新排列途径点，生成“不走回头路”的最优顺序。
2.  **可视化地图**:
    - 使用 Leaflet 加载 OpenStreetMap 数据。
3.  **真实道路轨迹**:
    - 不同于简单的直线连接，使用 **OSRM** 引擎绘制沿着真实街道行驶的蓝色轨迹。
4.  **自动地理编码**:
    - 输入中文地名（如“故宫”），自动调用 Nominatim API 转换为经纬度坐标。

## 🛠 技术栈
- **DeepSeek API**: 负责逻辑推理和行程排序。
- **Leaflet.js**: 渲染交互式地图。
- **Leaflet Routing Machine**: 调用 OSRM 服务进行路径计算和绘图。
- **Nominatim**: 免费的地点转坐标服务。

## 🚀 如何使用
1.  填写 DeepSeek 或 OpenAI 的 API Key。
2.  输入**起点**和**终点**。
3.  在文本框中列出所有想去的**途径点**（每行一个）。
4.  点击“智能规划路线”。
5.  等待几秒，地图上将自动连线，左侧会出现 AI 给出的交通建议 Summary。

## ⚠️ 注意事项
- 由于使用的是免费的 OSM 地理搜索接口，查询速度较慢（为了避免被封 IP，代码里加了 800ms 的延迟），请耐心等待 Loading 动画。
- 地名请输入官方全称（例如用“北京大兴国际机场”而不是“大兴机场”），识别率会更高。
